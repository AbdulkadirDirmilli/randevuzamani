# PWA Icons

Bu klasör PWA (Progressive Web App) ikonlarını içerir.

## Mevcut İkonlar

SVG formatında vektörel ikonlar kullanılmaktadır:

- `/favicon.svg` - Ana favicon (tüm boyutlar için)
- `/icons/icon.svg` - PWA manifest ikonu (512x512 viewBox)

## Avantajlar

SVG ikonlar kullanmanın avantajları:
- Sonsuz ölçeklenebilirlik
- Tek dosya ile tüm boyutlar
- Daha küçük dosya boyutu
- Kolay düzenleme

## Özelleştirme

İkon renklerini değiştirmek için SVG dosyalarındaki gradient değerlerini düzenleyin:
- `#7c3aed` - Ana mor renk
- `#a855f7` - Açık mor renk

## Uyumluluk

Modern tarayıcılar SVG ikonları desteklemektedir:
- Chrome 4+
- Firefox 3.6+
- Safari 5+
- Edge 12+
- Opera 10+
